Discuss – PHP Q&A Web Application:


📌 Project Description:

Discuss is a simple Question & Answer web application built using PHP, MySQL, HTML, CSS, and Bootstrap.
Users can register, log in, ask questions, answer questions, search questions, and manage their own posts.

This project is similar to a basic version of Stack Overflow where users can interact by asking and answering programming-related questions.

🚀 Features

User Registration and Login

Session-based Authentication

Ask a Question

View Questions

Answer Questions

Search Questions

Category-based Questions

Latest Questions Page

Delete Own Questions

Logout System
